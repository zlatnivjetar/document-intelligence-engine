**ExtractOptions interface:**
```typescript
import type { LanguageModelV3 } from '@ai-sdk/provider';
import { generateObject } from 'ai';
import { z } from 'zod';
import type { ExtractionInput, ExtractionResult } from './types.js';

export interface ExtractOptions<T> {
  input: ExtractionInput;
  schema: z.ZodSchema<T>;
  model: LanguageModelV3;
  schemaName?: string;
  schemaDescription?: string;
}
```

**Document encoding:** Convert the `input.document` to base64 string if it is a Buffer:
```typescript
const base64 = Buffer.isBuffer(input.document)
  ? input.document.toString('base64')
  : input.document;
```

**Confidence schema:** The confidence scores are requested from the LLM by augmenting the prompt. The LLM is asked to return a JSON object with two top-level keys: `extracted` (the schema-typed data) and `confidence` (a `Record<string, number>` object with one key per top-level field in the schema, each value between 0.0 and 1.0).

Build a combined Zod schema for the LLM response:
```typescript
const topLevelKeys = Object.keys((schema as z.ZodObject<z.ZodRawShape>).shape ?? {});
const confidenceSchema = z.object(
  Object.fromEntries(topLevelKeys.map((k) => [k, z.number().min(0).max(1)])),
);
const responseSchema = z.object({
  extracted: schema,
  confidence: confidenceSchema,
});
```

Note: The schema may not be a ZodObject (e.g., it could be a ZodRecord for flexible templates). Guard against this: if `(schema as z.ZodObject<z.ZodRawShape>).shape` is undefined, use an empty confidence schema `z.object({})` and derive keys from the result object at runtime.

**System prompt for the LLM:**
```
You are a precise document extraction assistant. Extract structured data from the provided document image or PDF exactly as it appears — do not infer or hallucinate values not present.

Return a JSON object with two keys:
- "extracted": the structured data matching the provided schema
- "confidence": an object with the same keys as "extracted", where each value is a number between 0.0 and 1.0 representing how confident you are in that field's value (1.0 = value is clearly present and unambiguous, 0.0 = value is absent or unreadable)
```

**User message content block:** Use Vercel AI SDK's `messages` format. The `content` array should include:
1. A text block: `{ type: 'text', text: 'Extract structured data from this document.' }`
2. A file/image block — use `type: 'file'` with `data` as the base64 string and `mimeType` from the input.

The AI SDK messages format for file content:
```typescript
{
  role: 'user' as const,
  content: [
    { type: 'text' as const, text: 'Extract structured data from this document.' },
    { type: 'file' as const, data: base64, mimeType: input.mimeType },
  ],
}
```

**generateObject call:**
```typescript
const result = await generateObject({
  model,
  schema: responseSchema,
  schemaName: schemaName ?? 'ExtractionResponse',
  schemaDescription: schemaDescription ?? 'Structured extraction with per-field confidence',
  messages: [userMessage],
  system: SYSTEM_PROMPT,
});
```

**Return value:** Compute overallConfidence as arithmetic mean of all confidence values:
```typescript
const confidenceValues = Object.values(result.object.confidence) as number[];
const overallConfidence =
  confidenceValues.length > 0
    ? confidenceValues.reduce((a, b) => a + b, 0) / confidenceValues.length
    : 0;
return {
  data: result.object.extracted,
  confidence: result.object.confidence,
  overallConfidence,
};
```
