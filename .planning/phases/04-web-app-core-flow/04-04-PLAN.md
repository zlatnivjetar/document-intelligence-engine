Create `apps/web/src/lib/pdf-inspect.ts` with these exact exports:
```typescript
export const MAX_PDF_INSPECT_SIZE_BYTES = 5 * 1024 * 1024;
export async function parsePdfInspectRequest(request: Request): Promise<{ file: File; buffer: Buffer }>;
export function createPdfInspectResponse(input: {
  filename: string;
  sizeBytes: number;
  pdfType: 'text-layer' | 'image-only';
  elapsedMs: number;
}): Response;
```

`parsePdfInspectRequest()` must:
- call `await request.formData()`
- read the `file` field
- reject missing files
- reject non-PDF mime types
- reject payloads over 5 MB
- convert `await file.arrayBuffer()` into `Buffer.from(arrayBuffer)`

Create `apps/web/src/app/api/pdf-inspect/route.ts` with:
```typescript
export const runtime = 'nodejs';
```

Implement `POST(request: Request)` so it:
1. Parses the request with `parsePdfInspectRequest()`
2. Captures start time with `performance.now()`
3. Calls `detectPdfType(buffer)` from `@docpipe/core`
4. Returns JSON with `filename`, `sizeBytes`, `pdfType`, and `elapsedMs`
5. Returns `400` with a JSON error message for invalid input

Do not store files on disk. Do not persist request data. This route is diagnostic only.
