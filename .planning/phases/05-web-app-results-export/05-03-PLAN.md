Create `apps/web/src/lib/result-export.ts` with these exact exports:
```typescript
import type { ExtractionResult } from "@/lib/docpipe";

export function downloadResultJson(fileStem: string, data: Record<string, unknown>): void;
export function downloadResultCsv(
  fileStem: string,
  result: ExtractionResult<Record<string, unknown>>,
): void;
export async function copyResultJson(data: Record<string, unknown>): Promise<void>;
```

Implement these concrete behaviors:
1. `downloadResultJson()` creates a `Blob` with `application/json;charset=utf-8`, uses `JSON.stringify(data, null, 2)`, and downloads it as:
   `{fileStem}-extraction.json`
2. `downloadResultCsv()` emits a CSV string with the exact header:
   `field,value,confidence`
3. `downloadResultCsv()` writes one row per top-level field in `result.data`
4. For each CSV row:
   - `field` = the top-level key
   - `value` = `JSON.stringify(value)` for objects/arrays, plain string conversion for scalars, empty string for `null`
   - `confidence` = the numeric confidence value from `result.confidence[key]` or empty string if missing
5. Escape every CSV cell by wrapping it in double quotes and doubling any internal `"` characters
6. Download the CSV file as:
   `{fileStem}-extraction.csv`
7. `copyResultJson()` writes `JSON.stringify(data, null, 2)` to `navigator.clipboard.writeText(...)`

Keep all three helpers browser-only. Do not add server routes or filesystem writes.
  </action>
